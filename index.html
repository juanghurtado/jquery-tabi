jQuery Tabi

Tabi is a jQuery plugin for tabbed interfaces. Usually, tabbed interfaces are difficult to manage, because the lack of horizontal space for the tabs. There are a few solutions around the web, and Tabi is just another one.

With Tabi you can define rows for the tabs so they can arrange themselves in the same way some desktop applications do. In this way, the tab you clicked remains in the last row of the group of tabs, near the content.
